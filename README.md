# 🛡️ Sentinel: End-to-End Customer Churn Prediction
This project demonstrates a full data science workflow to identify and predict customer churn for a subscription-based service.

## 📊 Key Features
- **Data Pipeline:** Automated cleaning and handling of missing values.
- **EDA:** Regional trend analysis identifying "High-Risk" months and cities (Medan, March).
- **Machine Learning:** Logistic Regression model with **76% Recall** for identifying leavers.
- **Business Action:** Automated export of "At-Risk" customers to Excel for the sales team.

## 🛠️ Tech Stack
- **Python:** Pandas, Scikit-Learn, Seaborn.
- **SQL:** Data extraction and city-level aggregation.
- **Excel:** Final reporting output.
